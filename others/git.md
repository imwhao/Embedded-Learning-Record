git 基本命令

| 命令                        | 作用                                 |
| --------------------------- | ------------------------------------ |
| git status                  | 查看仓库状态                         |
| git init                    | 初始化仓库                           |
| git add                     | 将文件加入缓冲区（此时未提交到仓库） |
| git commit                  | 将文件提交到仓库                     |
| git branch                  | 查看分支                             |
| git branch *branch* name    | 创建分支                             |
| git branch -d *branch name* | 删除分支                             |
| git merge                   | 分支合并                             |
| git log                     | 查看日志                             |
| git tag                     | 为当前分支添加标签                   |
| git checkout *branch name*  | 切换分支                             |
| git push                    | 将本地仓库push到远程仓库             |
| git pull                    | 将远程仓库pull到本地仓库             |
